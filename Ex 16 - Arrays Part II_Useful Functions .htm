Sorting an Array

The Array object's sort method sorts a given array in alphabetical order. To faciliate this JavaScript converts all Data Types to their String equivalent. If we want to sort a number array in real numeric order we have to apply a compare function.

The Array object's slice method is a simple way of making a new array from a series of elements found in another array. The arguments of slice method specify the beginning and the ending of the sequence of elements that you copy to the new array. If you specify as arguments negative numbers then the slicing is done from the end of original array.
