JavaScript Exercise 20 - Revision of Strings

Problem

Write a JavaScript program that checks if a given string is null or alphanumeric and if it is then it capitalizes the first letter of every word in this sentence.

Solution

We will test the script with 3 different strings. First a number string, second a null string and finally with a valid alphanumeric sentence
