Javascript Exercise 11 - Using the Date & Time Functions

Understanding Date

Date and Time manipulation are an essential part of JavaScript programming because they make it possible for Web applications to mark the passage of time. The Date data type includes both date and time values. Because dates and times come in great variety the Date object type includes many methods.

In the example below we'll see how we can use some of these.
