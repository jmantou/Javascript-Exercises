JavaScript Exercise 2 - Writing Content in Document

We can use the Document.write function to output text from JavaScript, but this method is most often used for testing purposes. However in a real Web app it's more common to use the 'innerHTML' property of the Document element. For testing and experimenting we can also use console.log() method that outputs text in the browser console.
