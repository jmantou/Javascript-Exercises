Disclaimer - The Math Object Functions

The Math Object is a special object in JavaScript Language. We use it to access math - related functionality, rather than create other objects. As with any other object, we can create an instance of the Math object. But usually we can use the appropriate property of the object without first instantiating a new one. For example if we want the value of pi, we can use this specific property of the Math object.
