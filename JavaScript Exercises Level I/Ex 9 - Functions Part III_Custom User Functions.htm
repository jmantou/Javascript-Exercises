Javascript Exercise 9 - Building Custom Functions

Disclaimer - Reducing Repetitive work by the use of Custom Functions

Generally, in every computer programming language, functions are used to reduce the work the programmer must perform when creating an application.

Functions help us reduce programming workload in the following ways:

  • Reusing the Code. By encapsulating a simple task in a function, we can reuse it as often as we want or need.
  • Ease of Understanding. Using functions helps the programmer reduce the complexity of the programming tasks.
  • Error and Bug Fixing. It is usually easier to trap bugs and errors when the program is correctly compartmentalized with functions.
  • Ease of Coding Changes. We can make changes in various places in the program by changing the function in only one place.

We will start with a simple function that takes one argument, which is a number and returns the factorial of this number.
